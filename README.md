# Ayurwiki - Ayurveda Knowledge Base

An open, community-driven knowledge base on Ayurveda, built with [MkDocs](https://www.mkdocs.org/) and the [Material](https://squidfunk.github.io/mkdocs-material/) theme.

Visit the live site at [ayurwiki.org](https://ayurwiki.org).

## About

Ayurwiki documents Ayurveda — the herbs, medicines, practices, yoga, and traditional wisdom that form one of the world's oldest systems of holistic health. The project was awarded **Gold** under the "Award for Excellence in Information Technology in AYUSH sector - 2019".

The site contains **5,400+** articles organized across these sections:

- **Herbs** — 2,200+ medicinal plants with properties, uses, and identification
- **Medicines** — Ayurvedic formulations and traditional preparations
- **Yoga** — Asanas, pranayama techniques, and mudras
- **Physiology** — Doshas, dhatus, srotas, and agni
- **Concepts** — Prakriti, rasa, guna, veerya, and more
- **Traditions** — Schools of thought and classical lineages
- **Manufacturers** — Ayurvedic pharmaceutical companies
- **Resources** — Books, literature, and journals

## Running Locally

```bash
# Install MkDocs Material
pip install mkdocs-material

# Start the development server
mkdocs serve
```

Open [http://localhost:8000](http://localhost:8000) in your browser.

## Contributing

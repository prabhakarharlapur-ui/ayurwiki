## Niyāmana Composition
| Pārada | - | - | 1 part |
| --- | --- | --- | --- |
| Phaṇi (nāga vallī) | - | Lf. | 1/16 part |
| Laśuna | - | Bl. | 1/16 part |
| Ambuja | - | Lf. | 1/16 part |
| Mārkava (Bhṛṅgarāja) | - | Pl. | 1/16 part |
| Karkoṭī | - | Fr. | 1/16 part |
| Ciṅcikā (cincā) | - | Lf. | 1/16 part |
| Kāṅjīka | - |  | 1/16 part |

## Method
[Kalka](Kalka.md) of ingradients 1 to 7 is prepared first.This is mixed with Kāṅjīka in a pot. Pārada is bundled in a Poṭṭali and Svadana is done by Dolā yantra for one day.

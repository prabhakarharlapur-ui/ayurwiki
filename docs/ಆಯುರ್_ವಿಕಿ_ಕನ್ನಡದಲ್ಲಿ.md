# ಆಯುರ್ ವಿಕಿ ಕನ್ನಡದಲ್ಲಿ

'ಆಯುರ್ ವಿಕಿ' ಯೋಜನೆಯ ಕನ್ನಡ ಆವೃತ್ತಿಗೆ ನಿಮಗೆ ಸ್ವಾಗತ. ಈ ಆವೃತ್ತಿಯನ್ನು ೨೦೨೨ರ ನವೆಂಬರ್ ತಿಂಗಳಿಂದ ಪ್ರಾರಂಭಿಸಿ ನಿಮ್ಮೆಲ್ಲರ ಮುಂದೆ ತರುವ ಪ್ರಯತ್ನ ನಮ್ಮದು. ಈ ಯೋಜನೆಗೆ ನೀವೂ ಕೈಜೋಡಿಸಬಹುದು. ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ ಅಥವ ಚರ್ಚೆಯನ್ನು ಪ್ರಾರಂಭಿಸಲು ಇದೇ ಪುಟದ 'Discussion' ಟ್ಯಾಬಿನಲ್ಲಿ ಒಂದು ಸಂದೇಶವನ್ನು ನೀವು ಸೇರಿಸಬಹುದು.

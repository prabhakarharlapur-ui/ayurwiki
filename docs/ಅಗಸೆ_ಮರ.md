# ಅಗಸೆ ಮರ

[TOC]

ಅಗಸೆ (ಅಗಚೆ) ಮರ.

## ಹುಟ್ಟು
ಉಲ್ಲೇಖ: ಸಹಸ್ರಾರ್ಧ ವೃಕ್ಷಗಳ ವರ್ಣನೆ

ತೋಟಗಳಲ್ಲಿ ಹುಟ್ಟು.

## ಉಪಯೋಗ
ಉಲ್ಲೇಖ: ಸಹಸ್ರಾರ್ಧ ವೃಕ್ಷಗಳ ವರ್ಣನೆ

ಉಪಯೋಗ: ಔಷಧವಾಗಿ ಉಪಯೋಗಿಸಬಹುದು: ಸಿಪ್ಪೆ ಶರೀರಕ್ಕೆ ಬಲ ಕೊಡುವುದು. ಅದರ ಕಷಾಯ ವಿರೇಚನ ಮಾಡುತ್ತದೆ, ಮೈಲಿಗೆ ರೋಗದಲ್ಲಿ ಸಹ ಉಪಯೋಗ. ಎಲೆಗಳು ಶ್ವರಕ್ಕೂ ಕಾಸಕ್ಕೂ, ಹೂವಿನ ರಸ ಮಂದ ಕಣ್ಣಿಗೂ ಒಳ್ಳೆಯದು.

## ಸಾಧಾರಣ ಉಪಯೋಗ
ಉಲ್ಲೇಖ: ಸಹಸ್ರಾರ್ಧ ವೃಕ್ಷಗಳ ವರ್ಣನೆ

ಎಲೆ ಹೂ ಕಾಯಿಗಳು ಪಲ್ಯ.

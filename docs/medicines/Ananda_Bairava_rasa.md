# Ananda Bairava rasa

[TOC]

**Ananda Bairava rasa** is Useful for Digestion, Indigestion, Flatulence, Expectorant, Fever, Dysentery and Blood dysentery.

## List of Ayurvedic herb in which used in this preparation

# Kāñcanāra Guggulu

[TOC]

**Kāñcanāra Guggulu** Vati is a preparation made with the ingredients in the Formulation composition given below with Guggulu as the basic ingredient.

## Formulation composition
| Kāñcanāra API | [Bauhinia variegata](Bauhinia_variegata.md) | St. Bk. | 480 g |
| --- | --- | --- | --- |
| Harītakī API | [Terminalia chebula](Terminalia_chebula.md) | P. | 96 g |
| Bibhītaka API | [Terminalia bellerica](Terminalia_bellerica.md) | P. | 96 g |
| Āmalakī API | [Phyllanthus emblica](Phyllanthus_emblica.md) | P. | 96 g |
| Śuṇṭhī API | [Zingiber officinale](../herbs/Zingiber_officinale.md) | Rz. | 48 g |
| Marica API | [Piper nigrum](Piper_nigrum.md) | Fr. | 48 g |
| Pippalī API | [Piper longum](Piper_longum.md) | Fr. | 48 g |
| Varuṇa API | [Crataeva nurvala](Crataeva_nurvala.md) | St. Bk. | 48 g |
| Elā (Sūkṣmailā API) | [Elettaria cardamomum](Elettaria_cardamomum.md) | Sd. | 12 g |
| Tvak API | [Cinnamomum zeylanicum](Cinnamomum_zeylanicum.md) | St. Bk | 12 g |
| Patra (Tejapatra API) | [Cinnamomum tamala](Cinnamomum_tamala.md) | Lf. | 12 g |
| Guggulu API -Śuddha | [Commiphora wightii](Commiphora_wightii.md) | O.R. | 996 g |

## Method of preparation
* Take all the ingredients of the pharmacopoeial quality.
* Wash, dry and powder the ingredients number 1 to 11 of the formulation composition separately and pass through sieve number 85, weigh them separately in the required quantities and mix.
* Crush weighed quantity of Guggulu - Śuddha, add fine powder of other mixed ingredients to it and pound well. Add Ghṛta to an extent required to facilitate the pounding and continue pounding till a semi-solid uniformly mixed mass of suitable plasticity is obtained.

## Description
* Spherical pills, black or brownish-black in colour, agreeable distinct odour and bitter taste.

## Storage
* Store in a cool place in tightly closed container, protect from light and moisture.

## Therapeutic uses
* Gulma (Abdominal lump), Gaṇḍamālā (Cervical lymphadenitis), Apacī (Chronic lymphadenopathy/scrofula), Graṇthi (Cyst), Vraṇa (Ulcer), Kuṣṭha (Diseases of skin), Bhagandara (Fistula-in-ano), Ślīpada (Filariasis).

## Dose
* 2-3 g daily in divided doses.

## Anupāna
* Mundadi Kvatha, Khadirasara Kvatha, Haritaki Kvatha, Hot water.

# Elaadi churna

[TOC]

**Elaadi churna** Useful for Indigestion, Cough, Hoarseness, Blood dysentery, Flatulence and Sore throat. Acts as a tonic for the heart.

## List of Ayurvedic herb in which used in this preparation

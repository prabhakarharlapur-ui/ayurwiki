# Jeerakaadyarishta

[TOC]

**Jeerakaadyarishta** is used to treat Indigestion, Dysentery, Tastelessness, Itching on hands and Legs, To increase breast milk, and Periods problem

## List of Ayurvedic herb in which used in this preparation
[Elettaria cardamomum](Elettaria_cardamomum.md), [Mesua ferrea linn](Mesua_ferrea_linn.md)

# Chyavanaprash

[TOC]

**Chyavanaprash** is used as a tonic for body. It will give strength to the body and It is used to treat Cough, Asthma, Tuberculosis and Urinary problems

## List of Ayurvedic herb in which used in this preparation
[Elettaria cardamomum](Elettaria_cardamomum.md), [Mesua ferrea linn](Mesua_ferrea_linn.md)

# Shatavari Rasayana

[TOC]

**Shatavari Rasayana** is Useful for Diseases related to the Uterus and For lack of milk in pregnant women, Useful in Urinary Tract Infections, Sperm enhancer and Energiser for men.

## List of Ayurvedic herb in which used in this preparation

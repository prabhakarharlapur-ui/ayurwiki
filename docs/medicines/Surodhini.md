# Surodhini

[TOC]

**Surodhini** is used to treat Diarrhoea, Dysentery, Vomiting, Nausea and Mouth ulcer

## List of Ayurvedic herb in which used in this preparation
[Madhuca longifolia](Madhuca_longifolia.md), [Ficus racemosa](Ficus_racemosa.md)

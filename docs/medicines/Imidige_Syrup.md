# Imidige Syrup

[TOC]

**Imidige Syrup** is used to treat loss of Apatite, Jaundice, Indigestion and Gallbladder problems.

## List of Ayurvedic herb in which used in this preparation
[Ocimum tenuiflorum](Ocimum_tenuiflorum.md)

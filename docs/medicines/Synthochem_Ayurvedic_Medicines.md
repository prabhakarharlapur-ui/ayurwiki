# Synthochem Ayurvedic Medicines

1. **Cough Syrup** -  Cough Syrup is used for the treatment of upper respiratory tract.

Composition

Syrup Each 15 ml contains extract of

* Cinnamonum iners (Tejpat): 2 gm
* T. Belerica (Bahera): 2gm
* Taxus Baccata (Talishpatra): 0.5 gm
* Alpinia Galanga ( Kulanjan): 0.5 gm
* Khurasani Ajwayan: 0.5
* D. Alba (Dhustur): 200 mg
* Solanum Jacquini (Kantkari): 1 gm
* Menthol: q.s.
* Sugar base: q.s

2. **Dentosol** - Dentosol is a paint used for treatment of throat troubles and in gum mouth.

Composition

Each 5 ml contains

* Camphor: 30mg
* Mentha Oil: 0.1 ml
* Thymol (Satajwain): 30 mg
* Clove Oil: 0.15 ml
* Kattha: 50 mg
* Syrup: 4.57 ml
* Amaranth red Color: q.s.

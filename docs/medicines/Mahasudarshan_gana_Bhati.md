# Mahasudarshan gana Bhati

[TOC]

![*MAHASUDARSHAN GHAN BATI*](../images/Small_1924974777.png)

## Importance
It is an ayurvedic anti-pyretic tablet effective in all type of fever like viral, Malaria fever etc.

## Dosage
1-2 tablets 3-4 times in a day.

## Indications
All types of fever (Anti-pyretic)

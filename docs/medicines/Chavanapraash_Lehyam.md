# Chavanapraash Lehyam

[TOC]

**Chavanapraash Lehyam** is Works as a tonic. Useful for cough, asthma and hoarseness.

## List of Ayurvedic herb in which used in this preparation
[Phyllanthus amarus](Phyllanthus_amarus.md), [Sesamum indicum](Sesamum_indicum.md), [Holarrhena pubescens](Holarrhena_pubescens.md)

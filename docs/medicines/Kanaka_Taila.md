# Kanaka Taila

[TOC]

**Kanaka Taila** is a medicated oil preparation made with the ingredients in the Formulation Composition given below with [murcchita](murcchita.md) Tila taila as the basic ingredient.

## Formulation composition
| Madhuka (Yaṣṭī API) | [Glycyrrhiza glabra](Glycyrrhiza_glabra.md) | Rt. | 768 g |
| --- | --- | --- | --- |
| Jala API for decoction | Potable Water | – | 3.072 ml |
| reduced to 768 ml |  |  |  |
| Taila (Tila Taila API) | [Sesame](Sesame.md) oil | – | 192 g |
| Priyaṅgu API | [Calicarpa macrophylla](Calicarpa_macrophylla.md) | Fl. | 6.4 g |
| Mañjiṣṭhā API | [Rubia cordifolia](Rubia_cordifolia.md) | St. | 6.4 g |
| Candana (Rakta Candana API) | [Pterocarpus santalinus](Pterocarpus_santalinus.md) | Ht. Wd. | 6.4 g |
| Utpala API | [Nymphaea stellata](../herbs/Nymphaea_stellata.md) | Fl. | 6.4 g |
| keṣara (Nāgakeṣara API) | [Mesua ferrea](Mesua_ferrea.md) | Stmn. | 6.4 g |

## Method of preparation
* Take all ingredients of pharmacopoeial quality.
* Treat Tila taila to prepare [mūrcchita](mūrcchita.md) Tila taila.
* Wash, clean, dry the ingredient number 1 of the Formulation Composition, powder and pass through 355 µm I. S. sieve (sieve number 44) ([kvātha dravya](kvātha_dravya.md)).
* Add water for decoction to the kvatha dravya and soak for 4 h, heat and reduce the volume to one-fourth. Filter through muslin cloth to obtain [kvātha](kvātha.md).
* Wash, dry the ingredients numbered 4 to 8 ([Kalka](../Kalka.md) dravya) of the Formulation Composition, powder separately and pass through 180 µm I. S. sieve (sieve number 85) to obtain fine powder.

## Description
* Medicated oil, reddish brown in colour with a faint odour.

## Storage
* Store in a cool place in tightly closed containers, protected from light and moisture.

## Therapeutic uses
* Mukha roga (Diseases of oral cavity), Vyaṅga (Local hyper pigmentation), Nīlikā (Mole).

## Dose
* External use for Abhyanga (apply on whole body)
* Nasya - 2 to 4 drops per nostril once or twice a day.

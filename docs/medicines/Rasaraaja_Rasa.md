# Rasaraaja Rasa

[TOC]

**Rasaraaja Rasa** is Useful for Dizziness, Insomnia. This medicine removes exhaustion from the Body and Invigorates the Nerves. Corrects blood pressure. Strengthens the heart.

## List of Ayurvedic herb in which used in this preparation

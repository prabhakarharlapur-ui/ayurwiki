# Stondab Syrup

![Stondab Syrup](../images/108-stondab-syrup.jpg)

**Dabur Stondab Syrup** is an Ayurvedic formulation which helps relieve urinary calculus and other urinary related symptoms. Be sure to re-cap it tightly after use.

## Key Ingredients
* Varuna
* Rakta Punarnava
* Guduchi
* Apamarga
* Pashanbheda

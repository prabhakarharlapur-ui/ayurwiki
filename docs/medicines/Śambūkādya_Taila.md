# Śambūkādya Taila

[TOC]

**Śambūkādya Taila** Vati is a preparation made with the ingredients in Formulation composition given below with [Guggulu](../medicines/Guggulu.md) as the basic ingredient.

## Formulation composition
| Kaṭu Taila API | [Brassica campestris](../herbs/Brassica_campestris.md) | Sd. Oil | 768 g |
| --- | --- | --- | --- |
| Kalka Dravya : |  |  |  |
| Śambūka | Pila globosa | Entire | 250 g |
| Jala | Water |  | 3 l |

## Method of preparation
* Take the raw materials of Pharmacopoeial quality.
* Clean live Apple Snail (Pila globosa) thoroughly with tap water, and remove the foreign matter.
* Boil Pila globosa with shell for 20 - 25 min.
* Wash and drain the excess water.
* Grind entire Śambūka to prepare [Kalka](../Kalka.md).
* Treat Sarsapa Taila to prepare Mūrcchita Sarṣapa Taila.
* Take Mūrcchita Sarṣapa Taila in a stainless steel vessel and heat it mildly.

## Description
* Medicated oil, yellow in colour, with strong, unpleasant odour.

## Storage
* Store in a cool place in tightly closed container, protect from light and moisture.

## Therapeutic uses
* Karṇagata nāḍīvraṇa (Abscess in ear).

## Dose
* 5-10 drops, for external use, once or twice a day.

## Anupāna
* Water.

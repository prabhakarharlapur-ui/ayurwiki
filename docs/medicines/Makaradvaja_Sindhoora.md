# Makaradvaja Sindhoora

[TOC]

**Chintamani Chaturmukha rasa** is An Aphrodisiac, Protects the body from diseases caused by germs. Useful for Indigestion, Dysentery, Heart diseases. Consuming it regularly leads to a long and healthy life.

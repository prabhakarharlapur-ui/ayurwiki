# Basti (Panchakarma)

Basti is treatment done with medicinal substances, like herbal oils and decoctions in a liquid medium, into the rectum of the person. This is one of the five Pradhana Karmas of [Panchakarma](../traditions/Panchakarma.md).

Basti is used to treat vata disorders.

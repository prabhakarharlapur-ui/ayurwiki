# Basant Kusumaakar Ras

[TOC]

**Basant Kusumaakar Ras** is used as Tonic for Heart and Brain. It is used to treat Daibetes, Kidney problems, Asthma, White discharge and Sexual disorders

## List of Ayurvedic herb in which used in this preparation

# Mahaakandardhaka Lehya

[TOC]

**Mahaakandardhaka Lehya** is Useful for Nausea, Vomiting, Indigestion, Flatulence and Bilious troubles

## List of Ayurvedic herb in which used in this preparation
[Zingiber officinale](../herbs/Zingiber_officinale.md), [Myristica fragrans](Myristica_fragrans.md)

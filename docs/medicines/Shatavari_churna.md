# Shatavari churna

[TOC]

**Shatavari churna** is used to treat Malabsorption syndrome, Piles, Abdominal Lump, Acts as Diarrhoea, Vatapitta and Rasayana. It will boost the memory power and Increases Breastmilk in Mothers

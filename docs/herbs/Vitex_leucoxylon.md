# Vitex leucoxylon

[TOC]

![](../images/Vitex_leucoxylon.jpg)
**Vitex leucoxylon** is a deciduous tree with a spreading crown growing up to 15 metres tall. It has a short, thick bole. The tree is harvested from the wild for medicinal use and as a source of timber. It is cultivated in India for medicinal use and is also grown as an ornamental plant.
## Uses
Intermittent fever, Catarrh, Headaches, Rheumatism.

## Parts Used
Leaves, Roots, Bark, Fruits.

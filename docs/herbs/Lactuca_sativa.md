# Lactuca sativa

[TOC]

![](../images/Lactuca_sativa_001.JPG)
**Lactuca sativa** is a Annual/Biennial up to 0.90 metres tall. It is harvested from the wild for local use as a food, medicine and source of materials.
## Uses
Cardiac paralysis, Warts, Insomnia, Anxiety, Neuroses, Dry coughs, Whooping cough, Rheumatic pain.

## Parts Used
Leaves, Seeds.

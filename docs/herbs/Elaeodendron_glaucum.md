# Elaeodendron glaucum

[TOC]

**Elaeodendron glaucum** is an evergreen tree. It grows upto 18 - 25 metres tall. The bole can be 40 - 95cm in diameter. The tree is harvested from the wild for local use as a medicine and source of wood. It is sometimes grown as an ornamental in gardens.
## Uses
Nerve diseases, Hysteria, Headaches, Swellings, Snakebite.

## Parts Used
Gum, Bark, Leaves.

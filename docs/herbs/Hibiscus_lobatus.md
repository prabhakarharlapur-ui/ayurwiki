# Hibiscus lobatus

[TOC]

![*Linaria vulgaris*](../images/Linaria_vulgaris_-_harilik_käokannus_Valingu.jpg)
**Hibiscus lobatus** is an annual herb, erect, 50-70cm in height, Branchlets hairy.
## Uses
Dysentery, Stomach ache, Sparmatorrhea, Anorexia.

## Parts Used
Roots, Leaves.

## Chemical Composition
The plant contains the cyclorpo, Methyl circulate, Methyl 2 hydroxysterculate, Malvalate and beta sterolate etc.

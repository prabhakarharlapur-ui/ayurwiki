# Euphorbia royleana

[TOC]

![](../images/Euphorbia_royleana.jpg)
**Euphorbia royleana** is an erect, shrub. It grows up to 3 metres tall. The main stem can be 40 - 50cm in diameter, producing many branches up to 7cm in diameter from its upper parts.

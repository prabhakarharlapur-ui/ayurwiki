# Tabernaemontana alternifolia - Maddarasa

[TOC]

![*Tabernaemontana alternifolia*](../images/Tabernaemontana_alternifolia_26.JPG)
**Tabernaemontana alternifolia** is small tree native to Western Ghats. It will grow upto 2-5 meter tall.
## Uses
Toothache, Wounds, Stye.

## Parts Used
Root, Flower bud, Milk.

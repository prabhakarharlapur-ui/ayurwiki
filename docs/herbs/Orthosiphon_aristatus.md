# Orthosiphon aristatus

[TOC]

![*Orthosiphon aristatus*](../images/Cat_whiskers_(Orthosiphon_aristatus)_(38891043775).jpg)
**Orthosiphon aristatus** is a perennial herb which grows upto 90cm tall. Another name is Java tea, which gives hints to both its native distribution (tropical Asia and Queensland) and how it is consumed as a medicine
## Uses
Kidney infection, Kidney stone, Poor renal function.

## Parts Used
Leaves.

# Geranium wallichianum

[TOC]

![](../images/Geranium_wallichianum_'Buxton’s_Blue’_04.JPG)
**Geranium wallichianum** is a perennial plant that can grow up to 0.30 metres tall.
It is harvested from the wild for local use as a medicine and source of materials.
## Uses
Peptic ulcers, Toothache, Bleeding, Joint pains.

## Parts Used
Root, Bark.

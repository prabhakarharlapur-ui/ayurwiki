# Ziziphus xylopyrus Willd - Ghoṇṭā

[TOC]

![*Ghoṇṭā*](../images/Raw_woody-fruited_jujube_fruit_(Ziziphus_xylopyrus)_at_Kambalakonda_Wildlife_Sanctuary.jpg)

**Ziziphus xylopyrus** is a straggling shrub distributed in North-West India, U.P., Bihar and South India, in moist deciduous forests.
## Uses
Headache, Acne, Pimples, Urinary disorders, Hysteria, Skin rashes, Stomachache, Diabetes, Bronchial asthma, Diarrhea, Excess thirst, Aphrodisiac.

## Parts Used
Leaves, Bark, Seeds, Roots.

## Chemical Composition
The pulp of the fruit contains reducing sugars, sucrose, citric acid, carotene, vitamin C and tannins.

# Onosma echioides

[TOC]

![](../images/Onosma_dalmatica_sl4.jpg)
**Onosma echioides** is a perennial plant that can grow up to 0.30 metres tall. It is harvested from the wild for local use as a medicine and source of materials.
## Uses
Rheumatism, Palpitations of the heart, Skin eruptions.

## Parts Used
, stem, leaves, Root.

## Chemical Composition
It contains mainly aliphatic ketones, lipids, naphthazarins, alkaloids, phenolic compounds, naphthoquinones

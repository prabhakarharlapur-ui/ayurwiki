# Ulmus wallichiana

[TOC]

![*Ulmus wallichiana*](../images/Ulmus_wallichiana_Bra51.png)
Ulmus wallichiana is a deciduous tree that can grow up to 35.00 metres tall.
The tree is harvested from the wild for local use as a source of materials. It is harvested on a commercial basis for its wood.

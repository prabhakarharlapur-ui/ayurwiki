# Viscum articulatum

[TOC]

![](../images/Viscum_articulatum_(2238694889).jpg)
**Viscum articulatum** is a much branched shrub. It can grow up to 90cm long.
## Uses
Fever, Bone dislocations, Kapha, Vata, Blood diseases, Ulcer, Epilepsy, Biliousness.

### Food
Viscum articulatum can be used in Food. Whole plant is roasted and consumed.

## Parts Used
Extract of the plant.

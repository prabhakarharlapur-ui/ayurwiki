# Lilium candidum

[TOC]

![](../images/Ab_plant_545.jpg)
**Lilium candidum** is a Bulb up to 1.00 metres tall. It is harvested from the wild for local use as a food, medicine and source of materials.
## Uses
Tumours, Ulcers, External inflammations, Epilepsy.

## Parts Used
Flowers.

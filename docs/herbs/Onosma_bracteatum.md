# Onosma bracteatum

[TOC]

**Onosma bracteatum** is a perennial plant that can grow up to 0.40 metres tall. It is harvested from the wild for local use as a medicine and source of materials.
## Uses
Rheumatism, Syphilis, Leprosy, Excessive thirst, Restlessness.

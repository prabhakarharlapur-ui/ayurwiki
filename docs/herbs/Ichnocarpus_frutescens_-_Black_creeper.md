# Ichnocarpus frutescens - Black creeper

[TOC]

![](../images/Ichnocarpus_frutescens_(8286148401).jpg)
**Ichnocarpus frutescens** is an much branched, Extensively twining shrub.
## Uses
Fever, Stomachache, Wounds, Vomit, To Improve Breast Milk.

## Parts Used
Root, Leaf.

## Chemical Composition
It contains the Various chemical category viz. Phytosterol, Triterpenes, Flavonoids and Various other phenolic compounds.

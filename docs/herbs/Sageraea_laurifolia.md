# Sageraea laurifolia

[TOC]

![](../images/Sageraea_laurifolia_(8257390982).jpg)
Sageraea laurina is an evergreen tree growing about 12 metres tall. The tree is harvested from the wild for local use as a medicine and source of wood. The plant is classified as 'Near Threatened' in the IUCN Red List of Threatened Species(2011).

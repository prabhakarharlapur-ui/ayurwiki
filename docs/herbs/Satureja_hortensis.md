# Satureja hortensis

[TOC]

![](../images/Satureja_hortensis_Prague_2011_1.jpg)
**Satureja hortensis** is an annual plant that can grow up to 0.30 metres tall. It is harvested from the wild for local use as a food, medicine and source of materials.
## Uses
Indigestion, Flatulence, Nausea, Diarrhoea, Bronchial congestion, Bronchial congestion, Menstrual disorders, Arthritic joints.

## Parts Used
Leaves, Flowering sho, leaves, Root.

## Chemical Composition
It contains Thymol, p-cymene, γ-terpinene and carvacrol were the main components of S. hortensis oil while thymol, γ-terpinene, and o-cymene etc.

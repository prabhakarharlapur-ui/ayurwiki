# Cerbera odollam - Auddalaka

[TOC]

![*Auddalaka*, *Cerbera odollam*](../images/Cerbera_odollum_-_IMG_1463.jpg)
**Auddalaka** is a small tree found along coasts in tropical countries. All parts of the tree yield a potent poison, especially the milky latex which copiously exudes from the stems.
## Uses
Skin diseases, Ringworm, Rabies

## Parts Used
Bark, Fruits, Leaves, Latex

## Identification
### Leaf
Simple, Alternate, clustered at ends, Petiole 8-35 mm long, slender, glabrous; lamina 10-25 x 2.5-6.5 cm, lanceolate or oblanceolate; base cuneate or attenuate.

### Flower
Bisexual, Axillary spiciform racemes, Pseudoterminal cymes, White, 5, 5 cm across, white, in pseudoterminal cymes; calyx lobes 5, linear, recurved, eglandular.

### Fruit
A drupe, Globose or ellipsoid, pericarp green turning rose; seeds 1 or 2 compressed.

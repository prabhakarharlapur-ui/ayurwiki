# Ipomea pes tigridis

[TOC]

**Ipomea pes tigridis** is a hairy annual which spreads on ground as hedges.
## Uses
Sores, Boils, Pimples, Tumours, Bronchial spasm, Haemoptysis, Dropsy.

## Parts Used
Mashed leaves, Seeds, Roots.

# Cymbopogon martinii

[TOC]

![](../images/Cymbopogon_martinii_Image.jpg)
**Cymbopogon martini** is an evergreen grass with numerous erect culms arising from a short, stout, woody rhizome. The stems are up to 3 metres tall.
## Uses
Rheumatism, Hair loss, Arthritis, Lumbago, Spasms.

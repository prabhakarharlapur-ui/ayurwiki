# Launaea sarmentosa

[TOC]

![*Linaria vulgaris*](../images/Linaria_vulgaris_-_harilik_käokannus_Valingu.jpg)
**Launaea sarmentosa** is a hairless, whip shaped, Procumbent, Perennial herb, Rooting and leaf bearing at the nodes and arching between them.
## Uses
Constipation, Soporific for children.

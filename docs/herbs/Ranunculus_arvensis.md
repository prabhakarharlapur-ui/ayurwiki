# Ranunculus arvensis

[TOC]

![](../images/Ranunculus_arvensis_sl16.jpg)
**Ranunculus arvensis** is an annual plant that can grow up to 0.50 metres tall. It is harvested from the wild for local use as a medicine.
## Uses
Intermittent fevers, Gout, Asthma.

## Parts Used
, stem, leaves, Root.

## Chemical Composition
It contains Aluminum chloride, ammonium molybdate, ascorbic acid (Vitamin-C), caffeic acid, catechin, dibasic sodium phosphate etc.

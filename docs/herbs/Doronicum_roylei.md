# Doronicum roylei

[TOC]

![](../images/Doronicum_kamaonense_(syn._Doronicum_roylei)_(7851332428).jpg)
**Doronicum roylei** is a perennial plant that can grow up to 1.20 metres tall.
It is harvested from the wild for local use as a medicine.
## Uses
Altitude sickness.

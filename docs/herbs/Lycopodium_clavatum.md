# Lycopodium clavatum

[TOC]

![](../images/Karukold.jpg)
**Lycopodium clavatum** is a Evergreen Fern up to 0.10 metres tall. It is harvested from the wild for local use as a medicine and source of materials.
## Uses
Kidney disorders, Rheumatic arthritis, Catarrhal cystitis, Gastritis, Skin diseases, Irritations, Bleeding noses, Injured tissues.

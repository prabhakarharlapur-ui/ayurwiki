# Mundulea sericea

[TOC]

![](../images/Mundulea_sericea_(6256612843).jpg)
**Mundulea sericea** is a branched shrub. Single stem with a bushy much branched crown and greenish yellow smooth bark.
## Uses
Poisoning, Infertility, Frequent miscarriages.

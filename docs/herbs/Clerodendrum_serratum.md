# Clerodendrum serratum

[TOC]

![](../images/Clerodendrum_serratum.jpg)
Clerodendrum serratum is a perennial woody shrub with blunt quadrangular stem.
## Uses
Common cold, Cough, Tuberculosis, Asthma, Chronic respiratory disorders, Headache, Earache, Wounds

## Parts Used

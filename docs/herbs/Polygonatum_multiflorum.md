# Polygonatum multiflorum

[TOC]

![](../images/20190422Polygonatum_multiflorum1.jpg)
**Polygonatum multiflorum** is a perennial plant that can grow up to 1.20 metres tall. It is harvested from the wild for local use as a food, medicine and source of materials.
## Uses
Stomach inflammations, Chronic dysentery, Pulmonary problems, Tuberculosis, Women's complaints, Bruises.

## Parts Used
Young shoots, Leaves.

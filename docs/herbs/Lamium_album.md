# Lamium album

[TOC]

![](../images/Flower_of_Lamium_album.jpg)
**Lamium album** is a perennial plant that can grow up to 0.60 metres tall. It is harvested from the wild for local use as a food, medicine and source of materials.
## Uses
Bladder complaints, Kidney complaints, Diarrhoea, Menstrual problems, Vaginal discharges, Kidney disorders, Amenorrhoea.

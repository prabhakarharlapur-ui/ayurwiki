# Fumaria officinalis

[TOC]

![](../images/Atlas_roslin_pl_Dymnica_pospolita_6577_7014.jpg)
**Fumaria officinalis** is an annual plant that can grow up to 0.30 metres tall. It is harvested from the wild for local use as a food, medicine and source of materials.
## Uses
Visceral obstructions, Scorbutic affections, Eczema.

## Parts Used
Flowers, stem, leaves, Root.

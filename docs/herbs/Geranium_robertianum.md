# Geranium robertianum

[TOC]

**Geranium robertianum** is a Annual/Biennial up to 0.40 metres tall. It is harvested from the wild for local use as a medicine and source of materials.
## Uses
Halt bleeding, Diarrhoea, Diabetes, Stomach ailments, Kidney infections, Jaundice, Painful breasts, Rheumatic joints.

## Parts Used
Leaves.

# Kingiodendron pinnatum

[TOC]

![](../images/Kingiodendron_pinnatum.jpg)
**Kingiodendron pinnatum** is an evergreen tree growing up to 30 metres tall. The bole can be up to 130cm in diameter. The plant is harvested from the wild for local use as a medicine and source of materials.
## Uses
Gonorrhoea, Catarrhal conditions, Respiratory tracts, Rheumatism.

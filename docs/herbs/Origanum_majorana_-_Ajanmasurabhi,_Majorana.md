# Origanum majorana - Ajanmasurabhi, Majorana

[TOC]

![](../images/Starr_070906-8859_Origanum_majorana.jpg)
**Origanum majorana** is a perennial plant that can grow up to 0.60 metres tall. It is harvested from the wild for local use as a food, medicine and source of materials.
## Uses
Indigestion, Painful menstrual cramps, Stomach cramps, Liver problems, Gallstones, Menopause symptoms, Nerve pain, Earache.

## Parts Used
Leaves, Root

## Chemical Composition
It contains origanol and origanol along with ursolic acid, oleanolic acid, β-sitosterol, and triacontanol.

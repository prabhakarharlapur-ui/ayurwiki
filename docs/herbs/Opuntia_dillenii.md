# Opuntia dillenii

[TOC]

![](../images/Opuntia_dillenii_-_Flickr_-_Juan_Ignacio_1976.jpg)
**Opuntia dillenii** is a shrubby, low-spreading to erect, evergreen cactus, sometimes forming large, wide clumps that are seldom more than 80cm tall.
## Uses
Diabetes.

## Parts Used
Fruits, Young pads.

## Chemical Composition
It contains ethyl, Dihydroxybenzoate, Dihydroxybenzoic acid, Quercetin, Methyl, Glucopyranoside.

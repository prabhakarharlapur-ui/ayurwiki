# Anchusa italica

[TOC]

![](../images/Anchusa_italica_(8765613422).jpg)
**Anchusa italica** is native to Italy. It is wonderfully statuesque plant with dense foliage and lance shaped mid green leaves. With spikes of rich gentian blue, giant forget-me-not type flowers, 'Dropmore' is a 1905 selection that is still available today.
## Uses
Coughing, Sores.

## Parts Used
Raw flowers.

## Chemical Composition
Chemical types include eight megastigmane compounds, five triterpenoid compounds.

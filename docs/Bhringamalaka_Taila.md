# Bhringamalaka Taila

**Bhringamalaka Taila** is prepared using:
* *[Eclipta alba](Eclipta_alba.md)* - Bhringa raja
* *[Phyllanthus emblica](Phyllanthus_emblica.md)* - Amalaki
* *[Tinospora cordifolia](Tinospora_cordifolia.md)* - Amrita
* *[Aegle Marmelos](Aegle_Marmelos.md)* - Bilwa
* *[Aloe Vera](Aloe_Vera.md)* - Kumari
* *[Cedrus Deodara](Cedrus_Deodara.md)* - Devadaru

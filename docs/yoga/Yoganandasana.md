# Yoganandasana

**Yoganandasana** is an Asana. It is translated as Pose Dedicated to Yogi Yogananda from Sanskrit.

The pose is described as a "standing one-legged balance, forward bend."

The name of this pose comes from "Yogananda" in reference to the Yogi, and "asana" meaning "posture" or "seat".

## Benefits and Cautions
This pose has the following benefits: it stretched the front of the body, promotes spinal flexibility and balance.

Be careful while doing this pose if you have any spinal injuries.

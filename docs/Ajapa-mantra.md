# Ajapa-mantra

Unconscious repetitie prayer. Every living creature unconsciously breaths the prayer 'Soham' (sah=he(the universal spirit), aham= am I) with each inward breath, and with each outgoing breath prays 'Haṁsah' (Aham=I am, Saḥ=he (the universal spirit)).

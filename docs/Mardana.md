# Mardana

[TOC]

**Mardana** is the process of trituration of drugs to a fine state of division with or without prescribed liquid.

## Mardana Composition
| Sūta (pārada) | - | - | 1 part |
| --- | --- | --- | --- |
| Guḍa | - |  | 1/16 part |
| Dagdhorṇa | - | - | 1/16 part |
| [Lavaṇa](Lavaṇa.md) (Saiṇdhava Lavaṇa) | - |  | 1/16 part |
| Mandira dhūma | - |  | 1/16 part |
| Iṣṭikā cūrṇa | - |  | 1/16 part |
| Āsurī (rājikā) | - | Sd. | 1/16 part |
| [Kāṅjīka](Kāṅjīka.md) | - |  | Qs. |

## Method
Inradients 2 to 7 are mixed with pārada. Kāṅjīka in required quantity is added and rubbed well for there days in a [kalka](kalka.md) dravyas are washed with warm or warm Kāṅjīka and is sqeezed through a thick cloth and the mercury is aeparated.

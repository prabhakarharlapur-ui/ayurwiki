# Ayurwiki.org:Introduction

Ayurwiki.org is a project created to document everything Ayurveda. While doing so, our focus would be to gather information that is distributed presently for easy access. We intend to make available information for easy recognition of herbs and make simpler the process of getting started with study or practise of Ayurveda.

You too can edit this wiki and help add information. For doing so, please register on the Wiki if you haven't yet. Once registered, start editing - start a new page or collaborate on any existing page.
